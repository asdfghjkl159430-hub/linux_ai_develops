# AI 工具使用声明（课程规范第十三条）

## 1. 允许使用的工具

本项目在开发过程中允许使用 ChatGPT / Copilot / Cursor Agent 等工具辅助：

- 生成 Shell 脚本骨架与注释模板  
- 生成 Markdown 文档结构与措辞润色  
- 排查 `awk`/`grep` 边界问题  

## 2. 标注说明

| 类别 | 说明 |
|------|------|
| 人工主导 | 路由关键词表、runbook 命令选择、sudo 策略、测试用例与课程对照表由成员审定 |
| AI 辅助 | 部分段落措辞、初版 `mermaid` 图、部分 `awk` 片段经 AI 建议后由成员手工验证 |
| 开源参考 | `bash`/`python` 通用写法参考官方手册与 POSIX 文档 |

## 3. 答辩承诺

全体成员能够：

- 解释 `agent-router.sh` 的意图匹配与退出码含义  
- 解释 `log_audit.sh` 管道顺序与 `AGENT_USE_SUDO` 分支  
- 现场修改一个关键词并演示路由变化  

若无法解释，将视为未掌握相关内容（按课程规范）。
